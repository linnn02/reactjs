## lab3.1
# описание 
тут реализован компонент StepCounter который показывает работу с пропс и стейт в реакт
компонент позволяет:
увеличиватьи уменьшать счетчик 
хранить историю изменений(последние 5 действий)
сбрасывать состояние
считать операции 

Props используются для передачи начального значения (initialValue) и шага  
State хранит текущее значение счётчика, историю изменений и количество операций 
Каждый экземпляр StepCounter имеет собственное состояние и работает независимо
# используемые технологии 
react vite
javascript
хуки
# запуск и установка
npm install
npm run dev
