## lab3.2
# описание 
здесь реализован компонент userprofile который загружает данные пользователя с внешнего апи и показывает работу хука useEffect
компонент позволяет:
загрузку данных
состояние ожидания(загрузка loading)
ошибки запроса

useEffect используется для загрузки данных при монтировании компонента и при изменении userId
AbortController позволяет отменять запрос при размонтировании компонента и при повторной загрузке
Dependency array в useEffect содержит userId, чтобы запрос выполнялся при его изменении

хуки

абортконтроллер
