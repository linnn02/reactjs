# Lab 10 Task 1
## MVC Todo List

Это приложение "Список задач", реализованное по паттерну **MVC** на JavaScript

### Возможности
- добавление задачи
- изменение статуса выполнения
- редактирование текста
- удаление задачи
- отображение статистики

### Структура
`TodoModel.js` хранение данных и CRUD операции
`TodoView.js` отображение интерфейса
`TodoController.js` связь между Model и View
`index.html` апуск приложения

### Как работает MVC
1. Пользователь выполняет действие в интерфейсе
2. View передаёт действие в Controller
3. Controller вызывает методы Model
4. Model обновляет данные и уведомляет View
5. View перерисовывает интерфейс

### Вывод
В этом задании реализован MVC-подход с разделением данных, интерфейса и логики
