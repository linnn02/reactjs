# Lab 10 Task 2
## MVVM User Profile Form
Это форма профиля пользователя, реализованная по паттерну **MVVM** на чистом JavaScript

### Возможности
- ввод имени, фамилии, email и возраста
- проверка данных в реальном времени
- вычисление полного имени
- сохранение и сброс формы

### Структура
`ViewModel.js` базовая реактивная логика;
`UserProfileViewModel.js` данные и валидация формы;
`UserProfileForm.js` интерфейс формы;
`index.html` запуск приложения.

### Как работает MVVM
1. View связана с ViewModel
2. Пользователь вводит данные в форму
3. ViewModel автоматически обновляется
4. Проверка и обновление интерфейса происходят сразу

### Вывод
В этом задании реализован MVVM-подход с реактивностью и двусторонней привязкой данных
