# Pattern Name
Email Validation Utility Pattern

## Description
Данный паттерн описывает стандартный подход к валидации 
email-адресов в JavaScript-приложениях с использованием utility-функции.

## Context Outline
Паттерн применяется в веб-приложениях, где требуется проверка 
пользовательского ввода перед отправкой данных на сервер.

## Problem Statement
Некорректный ввод email-адреса пользователем может привести к ошибкам 
в бизнес-логике приложения и ухудшению пользовательского опыта.

## Solution
1. Принять email-адрес в виде строки
2. Проверить соответствие формату email
3. Вернуть true, если email корректен, иначе false

## Design
Паттерн реализуется в виде чистой функции без побочных эффектов, 
что делает её легко тестируемой и повторно используемой.

## Implementation
Разработчик реализует функцию, принимающую cтроку и возвращающую логическое значение, 
используя регулярное выражение.

##  Examples
Пример использования приведён в файле pattern_example.js.

## Consequences
# Преимущества:
- Повышение качества данных
- Повторное использование
- Простота реализации

# Недостатки:
- Проверяется только формат, но не существование email
