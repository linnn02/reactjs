# Исследование Module Pattern

Module Pattern возник как способ организации кода и предотвращения
загрязнения глобального пространства имён в JavaScript.

Он активно использовался до появления ES6 модулей и применялся
во многих популярных библиотеках.

Revealing Module Pattern является вариацией Module Pattern,
которая делает публичный API более явным.

В современных приложениях Module Pattern эволюционировал
в ES6 modules, которые обеспечивают встроенную поддержку модульности.
