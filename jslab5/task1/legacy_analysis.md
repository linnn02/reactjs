### Анализ устаревшего кода

Обнаруженные проблемы:

1. Используется var вместо let/const.
2. Код находится в глобальной области видимости
3. Используются функции-конструкторы вместо классов
4. Нет модульной структуры
5. Все свойства публичные, нет private полей
6. Используется склеивание строк вместо template literals
7. Нет разделения логики на файлы
