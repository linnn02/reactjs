# Lab 8 Task 1: Flyweight Pattern

## Суть задания
В этом задании реализован паттерн **Flyweight** для уменьшения расхода памяти при создании большого количества похожих объектов

## Что сделано
`tree/FlyweightTree.js`  общий объект дерева
`tree/TreeFactory.js`  фабрика для кэширования деревьев
`forest/ForestRenderer.js`  генерация и отрисовка леса
`test/benchmark.js`  сравнение с обычным подходом
`index.html`  демонстрация

## Идея паттерна
Общие свойства дерева хранятся один раз:
- тип
- вид
- цвета
- размеры

Уникально для каждого дерева передаются:
- координаты
- масштаб
- поворот

## Результат
Сгенерирован лес из 10 000 деревьев с повторным использованием общих объектов, что уменьшает расход памяти
