# Lab 8 Task 2: Event Delegation

## Суть задания
В этом задании реализован паттерн **Event Delegation** для эффективной обработки событий у большого количества элементов

## Что сделано
`delegate/EventDelegator.js` единый обработчик событий
`todo/TodoList.js` список задач
`test/performance-test.js` сравнение производительности
`index.html` демонстрация

## Идея паттерна
Вместо множества обработчиков событий используется один обработчик на родительском контейнере

Это позволяет:
- экономить память
- упростить код
- автоматически поддерживать новые элементы

## Результат
Сделан todo list на 1000+ элементов с одной системой обработки событий
