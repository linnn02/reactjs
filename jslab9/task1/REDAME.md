# Лаб 9 Task 1
## Паттерн Observer / Pub-Sub

В данном задании реализован паттерн **Pub/Sub (Observer)** для имитации агрегатора новостей

### Основная идея
Издатель публикует новости по категориям, а подписчики получают уведомления только по тем темам, на которые они подписаны

### Структура проекта
`EventBus.js` центральная шина событий
`NewsPublisher.js` публикация новостей
`EmailNotifier.js` email уведомления
`PushNotifier.js` push уведомления
`DashboardWidget.js` виджет панели
`index.html` демонстрация

### Как работает
1. Подписчики подписываются на события через `EventBus`
2. Издатель публикует новость.
3. Все подписчики соответствующей категории получают уведомление

### Функционал
- подписка на события  
- публикация событий  
- отписка от событий  
- одноразовая подписка  
- фильтрация новостей по категориям  

### Вывод
Паттерн **Pub/Sub** позволяет организовать обмен событиями между компонентами без прямой зависимости между ними.
