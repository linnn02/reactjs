# Лаб 9 Task 2
## Паттерны Mediator и Command

В задании реализована система **умного дома** с использованием двух паттернов: Mediator и Command

### Структура проекта
`Device.js` базовый класс устройств
`Light.js` управление светом
-`Thermostat.js` термостат
`SecuritySystem.js` система безопасности
`HomeMediator.js` посредник между устройствами
`Command.js` базовый класс команды
`LightCommands.js` команды для света
`CommandManager.js` управление командами
`index.html` демонстрация

### Mediator
`HomeMediator` координирует взаимодействие устройств и получает уведомления об изменении их состояния

### Command
Команды (`TurnOnLight`, `TurnOffLight`, `SetBrightness`) выполняются через `CommandManager`, который хранит историю действий

### Функционал
- управление устройствами  
- выполнение команд  
- хранение истории действий  
- undo / redo  

### Вывод
Mediator упрощает взаимодействие устройств, а Command позволяет выполнять действия и отменять их
