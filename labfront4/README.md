# Лабораторная работа 4
## task1
## Обработка событий и валидация формы
В данной работе реализована форма регистрации пользователя на React

## Что реализовано
- поля ввода: имя, email, возраст  
- использование useState для хранения данных формы  
- обработка событий onChange  
- проверка данных при вводе  
- вывод сообщений об ошибках  
- обработка отправки формы через onSubmit  
- использование preventDefault для отмены перезагрузки страницы  
- вывод сообщения об успешной регистрации  
- очистка формы после успешной отправки  

## Как работает форма
При вводе данных выполняется проверка:
- имя должно содержать минимум 2 символа  
- email должен быть в корректном формате  
- возраст должен быть числом и не меньше 18  

Если данные введены неверно, под полем появляется сообщение об ошибке 
Если все данные введены правильно, появляется сообщение об успешной регистрации

## Что было изучено
В ходе выполнения работы я разобралась с controlled components и научилась использовать preventDefault  



## task2  
## Разделение компонентов React
В данной работе реализовано приложение для добавления и отображения статей

## Что реализовано
Приложение состоит из компонентов:
- ArticleManager основной компонент  
- AddArticle форма добавления статьи  
- ArticleList список статей  
- ArticleItem отдельная статья  

Состояние списка статей хранится в ArticleManager
Состояние раскрытия статьи хранится внутри ArticleItem с использованием useState

## Что было изучено
В ходе выполнения работы я разобралась с использованием useState в дочерних компонентах
